Redesigned Telavox's softphone application with a sleek, modern interface, adding RTL support for Arabic users. The project involved creating a more intuitive user experience while maintaining all the powerful communication features that Telavox users rely on.

Key achievements:
- Modern, clean interface design
- RTL (Right-to-Left) language support for Arabic users
- Improved user experience and accessibility
- Cross-platform compatibility with Electron
