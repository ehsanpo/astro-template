# Image Placeholders

This folder contains placeholder files for the portfolio images. Replace these with actual screenshots/images of your Bookmark Library app:

## Required Images:

- `bookmark-chaos.jpg` - Background image showing bookmark chaos or the app interface
- `bookmark-icon.png` - App logo/icon
- `app-overview.jpg` - Main app interface screenshot
- `category-view.jpg` - Category browsing interface
- `search-feature.jpg` - Search functionality screenshot
- `upload-process.jpg` - File upload interface

## Image Guidelines:

- Background image should be high resolution (1920x1080 or similar)
- Logo should be square format (512x512 recommended)
- Screenshots should be clean and well-lit
- Consider using mockups or device frames for mobile screenshots
